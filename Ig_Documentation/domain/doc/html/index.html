IGUANA provides automatic documentation procedure based on doxygen.

Perl scripts generate the IGUANA documentation in a few steps:

  • Step 1: Create Doxyfile to configure doxygen;
  • Step 2: Preprocess project repository to generate additional doxygen pages;
  • Step 3: Run doxygen;
  • Step 4: Clean up.

    The documentation is generated by typing scram b doc command withing scram development area. Here is a definition of scram tag doc: 

    doc :
	perl $(LOCALTOP)/src/Ig_Documentation/IgDoxygen/src/createDcfg.prl $(SCRAM_PROJECTNAME) $(SCRAM_PROJVERSION) $(LOCALTOP)/src $(LOCALTOP)/src/Ig_Documentation/IgDoxygen/doc/html $(LOCALTOP) ;
	perl $(LOCALTOP)/src/Ig_Documentation/IgDoxygen/src/preDoxygen.prl $(SCRAM_PROJECTNAME) $(SCRAM_PROJVERSION) $(LOCALTOP)/src $(LOCALTOP)/src/Ig_Documentation/IgDoxygen/doc/html $(LOCALTOP)/myDoc ;
	cd $(LOCALTOP) ; \
	doxygen; \
	rm Doxyfile ; \
	rm -r ./myDoc ;
