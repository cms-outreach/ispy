IGUANA Overview

IGUANA is a modular C++ toolkit for interactive visualisation. It covers three domains: graphical user interfaces, interactive detector and event visualisation, and interactive data analysis and presentation. We use suitable existing software where possible, and integrate the whole in a coherent fashion. Application developers can select from the toolkit only those parts which are relevant for a particular application. We currently support IGUANA on Linux, Solaris and Windows; porting to other unix varieties is reasonably straightforward. IGUANA development is mainly driven by the needs of the CMS experiment at the LHC, but most of IGUANA is independent of CMS and may be freely used by others.
