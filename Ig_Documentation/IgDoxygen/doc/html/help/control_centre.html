Control Centre/Categories


Control Centre:

Control centre is to host different control categories e.g. Clip planes, Slicers, Lights, View Properties etc. Clicking on either toolbar button or "File"->"Control centre" menu the control centre of IGUANA will pop up. There are five buttons on it
  • Back: This is to go one level back/up. On the top level of the Control centre this button will be disabled.
  • Close: To close to IGUANA control centre. When you close the control centre and re-open it from either the toolbar button or "File"->"Control centre" menu, the control centre will show the last select category/icon set.
  • Revert: To revert the changes made via GUI(Control Category GUI section) which are not yet applied. When track changes option on the control category is not available or if available and enabled then the changes made on the GUI will automatically be applied. Normally this button is disabled and become enabled when fields on the control category have changed and not yet applied.
  • Apply: To apply the changes made via the GUI (Control category GUI section) which are not yet applied. When track changes option on the control category is not available or if available and enabled then the changes made on the GUI will automatically be applied and one can not revert those changes. Normally this button is disabled and become enabled when fields on the control category have changed and not yet applied.
  • Help: To get the help, this is disabled currently.
One can right click on any part of the control centre or selected category and then selecting "Whats This" from the popup menu to get quick help on that.

Control Categories:

Control Categories provides graphical interface for different objects which you want to control. There are many different types of control categories
  1. Some categories can keep track the changes made via GUI and provide functionality to apply or revert those changes on demand or automatically. You will see a "Track changes" checkbox at the end of these type of categories. If "Track changes" check box is available and checked then changes made on the GUI will automatically be applied otherwise "Apply" and "Revert" buttons of the control centre will become enabled and now you can apply or revert these changes.
  2. Some categories do not have "Track changes" checkbox. For these type of categories you can not revert the changes made on the GUI. Changes will be applied automatically and "Apply" and "Revert" buttons on the control centre will always remain disabled.
  3. Some categories can manage multiple objects of the same type e.g. Clip planes, Lights, Slicers etc. are the category which can manage multiple objects. You will see a Combo box at the top of these types of categories and by clicking on the small arrow icon on the right of this box will drop down the box and show you the available items. You can select any item from this list.
  4. Some categories of type 3 also allow you to add/delete items. You will see two small buttons with "+" and "x" sign on them. By Clicking on "+" button will add a new item and "x" will delete the currently selected item.
When you open the control centre, you will see different icons for different categories. By double clicking on any of these categories icons will show you the controls for that category. Below are the details about different control categories.

Clip Planes Category:

It is a control category to manipulate/add/delete clip planes. This category belongs to type 1 and 4 of the control categories types. To manipulate/add/delete a clip plane, open the control centre and then double click on the "Clip Planes" icon or select "Clips" from the "View" menu. This category is only available for 3D views.You can
  • select a different object from the combobox at the top of this category.
  • add/delete a objects using the "+" and "x" buttons.
  • enable/disable the track changes by checking/unchecking the "Track changes" checkbox.
  • activate/de-activate the selected object by checking/unchecking the "Active" checkbox.
  • show/hide the manipulator of the object on the scene graph by checking/unchecking the "Manipulator" checkbox. Manipulators are another way of manipulating the objects. To manipulate the object via manipulator, you have to change the to the "Pick/Select" mode by clicking on the toolbar button.
  • change the distance of the clip plane from origin by changing the "distance" field on this category.
  • change the normal of the clip plane by changing the xyz-axis fields on this category or align the normal to x, y or z axis by clicking on buttons on this category.

Slicers Category:
 
It is a control category to manipulate/add/delete slicers. Slicers are two back to back clip planes to show a slice of the scene graph. This category belongs to type  1 and 4 of the control categories types. To manipulate/add/delete a slicer, open the control centre and then double click on the "Slicers" icon or select "Slicers" from the "View" menu. This category is only available for 3D views. You can
  • select a different object from the combobox at the top of this category.
  • add/delete a objects using the "+" and "x" buttons.
  • enable/disable the track changes by checking/unchecking the "Track changes" checkbox.
  • activate/de-activate the selected object by checking/unchecking the "Active" checkbox.
  • show/hide the manipulator of the object on the scene graph by checking/unchecking the "Manipulator" checkbox. Manipulators are another way of manipulating the objects. To manipulate the object via manipulator, you have to change the to the "Pick/Select" mode by clicking on the toolbar button.
  • change the distance of the centre of the slicer from origin by changing the "distance" field on this category.
  • change the thickness of the slicer by changing the "Thickness" field on this category.
  • change the normal of the slicer by changing the xyz-axis fields on this category or align the normal to x, y or z axis by clicking on buttons on this category.
Animators Category:

It is a control category to manipulate/add/delete animations to the scene graph. Currently there are only fixed animations around x, y and z-axis are available. This category belongs to type  1 and 4 of the control categories types. To manipulate/add/delete a animator, open the control centre and then double click on the "Animators" icon or select "Animations" from the "View" menu. This category is only available for 3D views. You can
  • select a different object from the combobox at the top of this category.
  • add/delete a objects using the "+" and "x" buttons.
  • enable/disable the track changes by checking/unchecking the "Track changes" checkbox.
  • Start/stop the animation by check/unchecking the "Start/Stop" checkbox.
  • change different fields e.g. maximum time, time step, frequency by changing the "Max Time", "Time Step" and "Frequency" fields on the category.
  • change the axis of the animation by changing the xyz-axis fields on this category or align it to x, y or z axis by clicking on buttons on this category.
Lights Category:

It is a control category to manipulate/add/delete lights. This category belongs to type  1 and 4 of the control categories types. To manipulate/add/delete a light, open the control centre and then double click on the "Lights" icon or select "Lights" from the "View" menu. This category is only available for 3D views. You can
  • select a different object from the combobox at the top of this category.
  • add/delete a objects using the "+" and "x" buttons. You can not delete the "Head Light".
  • enable/disable the track changes by checking/unchecking the "Track changes" checkbox.
  • turn on/off the selected object by checking/unchecking the "On" checkbox.
  • show/hide the manipulator of the object on the scene graph by checking/unchecking the "Manipulator" checkbox. Manipulators are another way of manipulating the objects. To manipulate the object via manipulator, you have to change the to the "Pick/Select" mode by clicking on the toolbar button.
  • change the type of the light by selecting a different type from the 2nd combobox. Available types are "Spot Light", "Point Light" and "Directional Light".
  • change different fields of the light e.g. color, intensity, direction, location(only for spot lights), drop-off rate (only for spot lights), cut-off rate (only for spot lights).
Viewpoints Category:

It is a control category to manipulate/add/delete view points. This category belongs to type  1 and 4 of the control categories types. To manipulate/add/delete a view point, open the control centre and then double click on the "Viewpoints" icon or select "Viewpoints" from the "View" menu. This category is only available for 3D views. You can
  • select a different object from the combobox at the top of this category.
  • add/delete a objects using the "+" and "x" buttons. You can not delete the "Current Viewpoint".
  • enable/disable the track changes by checking/unchecking the "Track changes" checkbox.
  • turn on/off the auto clipping by checking/unchecking the "Auto Clipping" checkbox. when auto clipping is one then the "Near Clip Plane" and "Far Clip Plane" fields are disabled otherwise enabled.
  • rotate the camera along x, y and z axis by changing "Rotate X", "Rotate Y" and "Rotate Z" fields. You can align the camera along x, y and z axis by clicking on "Align" buttons on this category.
  • change the focal distance and aspect ratio of the camera via the "Focal Distance" and "Aspect Ratio" fields.
  • zoom in/out the camera.
View Properties Category:

It is a control category to edit different properties of Examiner Viewer, 3D Scene Manager, GL Render, Qt and Printing. This category belongs to type 2 of the control categories types. To edit these properties, open the control centre and then double click on the "View Properties" icon or select "Print/View Setting" from the "View" menu. This category is only available for 3D/2D views. From "Examiner Viewer" section on this category, you can
  • change different setting for the examiner viewer e.g. "Still Draw Style", "Interactive Draw Style", "Render Buffer Type",  "seek Time" etc.
  • enable disable "Auto Clipping", "Viewing", "3D Animation", "Popup menu", "Mouse cursor" etc.
From the "3D Scene Manager" section, you can
  • change the "Background Color"
  • enable disable "RGB Mode" and "Real Time update"
From the "GL Render" section, you can
  • change the "Transparency Type", "Redraw Properties", "Antialiasing Passes"
  • enable/disable "Auto Redraw", "Render Pass update"
From the "Qt Properties" section, you can
  • select the style of the application e.g. "Windows", "Motif", "SGI" etc.
From the "Print Options" section, you can
  • enable/disable various optiosn for the high quality vector printing e.g. "Simple Line Offset", use of "Best Root" finding, "Occlusion Culling", "Back ground" printing, "Land scape orientation" etc.
Node Manipulator Category:

It is a control category to manipulate a SoNode. This category belongs to type  1 of the control categories types. To manipulate a node, first you have to pick/select a node either by selecting the twig from the twig browser by clicking on a twig or by picking a 3D node from the 3D scene graph (in "Pick/Select Mode"). Once you have picked/selected a node, now to manipulate it, you can either open control centre and then select "Node Manipulator" category or click on the "Node Manipulator" menu item on the "View" menu. Node manipulator category will automatically be populated by the all the controllable fields. If no node is selected then this category will be empty.

2D View Category:

It is a control category to manipulate the camera for 2D view. This category belongs to type 2 of the control categories types and only available for 2Dviews. To manipulate camera for 2D views, open the control centre and then double click on the "2D View" icon or select "2D View properties" from the "View" menu. You can
  • Align camera to x, y and z axis by clicking on "Align Camera" buttons on this category.
