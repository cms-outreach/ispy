Viewer Buttons

If the SceneViewer window is too small, not all of these buttons will be visible.

Control Centre of IGUANA

Display the control centre. From Control centre of IGUANA you can manipulate with Lights, Clip Planes, Slicers, Animators, View Properties, Viewer Properties, Print Properties, Node manipulation etc.

  Zoom In/Out

Zoom in to or out of the scene graph.

Select/Pick Button

Selects object manipulation or pick mode (and deselects camera or viewer mode). The cursor shape will change to an arrow. In this mode, the user is manipulating objects in the scene graph. Seek button will be disable in this mode.

View Button

Selects camera or viewer mode (and deselects object manipulation or pick mode). The cursor shape will change to two half circle arrows. In this mode, the user is moving the camera in 3D space.

Home Button

Returns the camera to its home position (initial position if not reset).

Set Home Button

Resets the home position to the current camera position.

View All Button

Brings the entire scene graph into view.

Seek Button

Allows the user to select a new center of rotation for the camera. When clicked on (and in viewer mode) the cursor changes to a crosshair. The next left mouse buttonpress causes whatever is underneath the cursor to be selected as the new center of rotation. Once the button is released, the camera either jumps or animates to its new position depending on the current setting of the seek time in the preferences dialog box.

Camera Alignment Buttons

Select the axis of alignment (Z, Y, or X) of the camera. Only displayed for Plane Viewers.

Show/Hide Feedback axis
Show/Hide feed back axis on the scene graph.
Projection Button

Selects the type of camera used by the viewer. It toggles between the two available camera types --perspective and orthographic. Only displayed for Examiner and Plane viewers.

Invert Camera
Invert the camera i.e rotate the camera 180 degree anti-clockwise.
