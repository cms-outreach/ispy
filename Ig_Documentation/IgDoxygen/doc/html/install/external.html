External software packages

IGUANA makes use of a number of external software packages. The complete list of external packages and their versions needed to configure and build this version of IGUANA (@VERSNUM@) can be obtained from SCRAM configuration explorer.

The above SCRAM page also allows you to download and install each package. For convenience, a number of the packages are available in a single distribution bundle provided by the IT/API (Anaphe) group. If you are a member of the CMS collaboration, you may wish to download the CD-ROM image containing all CMS software (binaries and source) together with the external packages from the ORCA page.

The toolkit nature of IGUANA means that you are not obliged to install all possible external packages prior to using IGUANA. The lack of a particular external package will merely reduce the available IGUANA functionality. For example, if you do not with to install any external commercial packages you can still make use of much of the IGUANA toolkit. Some of the commercial components have free alternatives you wish to use:

  • TGS OpenInventor can be replaced by Coin3D
  • Native OpenGL can be replaced by Mesa

The package dependencies page can help you to see which IGUANA packages require which external software, as well as giving a wealth of other information about IGUANA organisation.
