Prebuilt IGUANA releases

Now one can download the pre-build RPMs of IGUANA/IGUANACMS using the XCMSI (CMS Software installation tool). Here is what you need to install IGUANA/IGUANACMS

    Lets say, you want to install IGUANA_x_y_z on your machine in /opr/cms directory:

    • setenv VO_CMS_SW_DIR /opt
    • cd /home/user/xyz
    • wget http://cmsdoc.cern.ch/cms/oo/repos_standalone/download/download/download.sh
    • chmod a+x download.sh
    • ./download.sh IGUANA_x_y_z
    • cd DVD-x.y
    • ./xcmsi.pl

    Now the XCMSI GUI will appear and you just have to do:

    • Click on the "Select Tags" button and select the IGUANA_x_y_z tag from the newly opened window and click on "Select" botton.
    • Click on "Start Installation" button if this is first time you are using XCMSI to install any CMS software otherwise click on "Update Installation" button.
    • Once installation is done, you can now use IGUANA by changing directory to the /opt/cms/Releases/IGUANA/IGUANA_x_y_z and setting up the IGUANA runtime environment by running the "scram runtime" commnad.

    It will take some time to install the RPMs. For more detail documentation on XCMSI please goto the XCMSI web site. If you have any questions/problems about the XCMSI usage then please send email at cms-xcmsi-feedback@cern.ch or submit a bug report on LCG Savannah
