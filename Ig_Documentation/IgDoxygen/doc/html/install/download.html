Installation via SCRAM

IGUANA is a SCRAM-managed project. You can get the list of locally installed versions of IGUANA by issuing the command scram list. If scram command is unknown, make sure that SCRAM is installed and your environment is properly set up. Following are some steps one can do to install scram

Set CVSROOT, for tcsh/csh
> setenv CVSROOT \
:pserver:anonymous@cmscvs.cern.ch:/cvs_server/repositories/SCRAM

and for sh/bash
> export \
CVSROOT=:pserver:anonymous@cmscvs.cern.ch:/cvs_server/repositories/SCRAM

and then
> mkdir SCRAM
> cd SCRAM
> cvs login #cvs password is 98passwd
> cvs co -d <scram_version> -r <scram_version> SCRAM
> cd <scram_version>/Installation
> ./install_scram

install_scram script will ask you two questions about the site name (e.g. FNAL ) and the default tool file for your site (e.g. tool-FNAL.conf, make sure the tool file must end with <sitename>.conf)

If you have SCRAM but the list command shows no IGUANA releases, you need to download and build IGUANA using SCRAM. This builds IGUANA from source, so you must first ensure that the required external softwares are installed for your system. Here are some quick commands one can use to download, build and install IGUANA using SCRAM

Set CVSROOT, for tcsh/csh
> setenv CVSROOT \
:pserver:anonymous@cmscvs.cern.ch:/cvs_server/repositories/IGUANA

and for sh/bash
> export \
CVSROOT=:pserver:anonymous@cmscvs.cern.ch:/cvs_server/repositories/IGUANA

and then
> mkdir IGUANA ; cd IGUANA
> cvs login #cvs password is 98passwd
> cvs co -p -r @LAST_VERSION@ \
Ig_Infrastructure/IgBuildScripts/src/make-release > make-release
> sh ./make-release @LAST_VERSION@

Now you are ready to install IGUANA in your local SCRAM installation. For that do
> cd @LAST_VERSION@
> scram install IGUANA @LAST_VERSION@

You can find more information about scram here.

The CVS repository

In addition to downloading IGUANA via SCRAM, you can also access our CVS server directly (you will still need SCRAM to build IGUANA). You would normally want to create an empty project area using SCRAM, and then fetch the IGUANA source into it. You may also wish to take a look at the make-release script which dowloads and optionally builds a release.

For access to the CVS repository, please refer to the CMS CVS server documentation. The project name is IGUANA; the tag for this release is @LAST_VERSION@ (see releases page for the tags of the other releases).
