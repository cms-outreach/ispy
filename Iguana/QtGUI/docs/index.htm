iSpy a powerful and lightweight CMS event display

Download iSpy from this page then double-click to start viewing real or simulated CMS events served from the Web

Features

  • Standalone iSpy viewer
    No need to install CMS or other software
  • Web access to real CMS Data
    Produced continuously by CMS online
  • Powerful, light and fast
    Small executable, starts and runs fast
  • Access all CMS data tiers
    Raw data, MC, physics objects
  • Fully interactive event data viewer
    Powerful 2D/3D graphics display
  • Create your own files for iSpy
    Using our iSpy CMSSW analysers
