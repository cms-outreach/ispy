iSpy How to make .ig files for iSpy using CMSSW

The following applies to CMSSW version 3_1_x or later

iSpy reads "ig files" containing event and detector data. These are created by running standard CMSSW analyzers, as follows:

 

1) Set-up your environment:

For csh:

setenv CMSSW_VERSION CMSSW_3_3_X

For sh:

export CMSSW_VERSION=CMSSW_3_3_X
2) Check out sources and build:
project CMSSW
cmsrel $CMSSW_VERSION
cmsenv
addpkg ISpy/Analyzers V01-01-02
scram b
3a) Run the test analyzer to make a Monte-Carlo .ig file:
cmsenv
cmsRun ISpy/Analyzers/python/ispy_33X_cfg.py
3b) Run the test analyzer to make an .ig file from real data:
cmsenv
cmsRun ISpy/Analyzers/python/ispy_Collisions09_cfg.py
3c) Use iSpy with your custom configuration file:
You only need to add the iSpy path to your process: 
process.load("ISpy/Analyzers/ISpy_Producer_cff")
process.p1 = cms.Path(process.iSpy_sequence)
